# Contributing to IHL Kernel

Thank you for your interest in contributing to the IHL Kernel project!

## How to Contribute

### Reporting Issues
- Open an issue describing the problem or suggestion
- Include relevant context and references to IHL sources

### Submitting Changes
1. Fork the repository
2. Create a new branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Commit with a clear message (`git commit -m "Add: description"`)
5. Push to your branch (`git push origin feature/your-feature`)
6. Open a Pull Request

## Content Guidelines
- All contributions must be grounded in established International Humanitarian Law sources
- Cite your sources using accepted legal citation formats
- Use clear, neutral, and precise legal language
